### Setup .env file on backend

```bash
MONGO_URI=your_mongo_uri
PORT=5000
JWT_SECRET=your_secret_key
CLIENT_URL= http://localhost:5173
```

### Run backend locally

```shell
npm run dev
```

### Setup .env file on frontend

```shell
VITE_BASE_URL=your_backend_url
```

### Run frontend locally

```shell
npm run dev
```
